## User Flow

### Clicking a Button
1. User clicks button (e.g., "View Room Rates")
2. Button becomes grayed out (disabled)
3. Loading message appears: "Loading rates..."
4. Wait for 1.5 seconds (thread is working)
5. Message box pops up with information
6. User clicks OK on message box
7. Loading message disappears
8. Button becomes clickable again

### Visual States

#### State 1: Ready (Default)
- All buttons enabled (blue, clickable)
- Loading label invisible
- User can click any button

#### State 2: Loading (Button Clicked)
- Clicked button disabled (grayed out)
- Other buttons still enabled
- Loading label visible and orange
- Thread running in background

#### State 3: Showing Information
- Message box displayed
- Button still disabled
- Loading label still visible
- Waiting for user to click OK

#### State 4: Back to Ready
- All buttons enabled again
- Loading label invisible
- Ready for next click

## Dimensions Reference

- **Form**: 800 x 500 pixels
- **Logo PictureBox**: 150 x 150 pixels
- **Each Button**: 180 x 45 pixels
- **GroupBox**: 720 x 120 pixels
- **Margins**: Usually 20-30 pixels

## Font Sizes

- **Title**: 24pt (biggest)
- **Subtitle**: 14pt
- **Buttons**: 11pt
- **Description**: 10pt
- **Loading**: 10pt
- **GroupBox Header**: 10pt
